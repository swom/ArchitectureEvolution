# ArchitectureEvolution

## Peregrine

```bash
module load CMake
module load binutils
```

## All platforms
```
git clone git@github.com:swom/ArchitectureEvolution.git
cd ArchitectureEvolution
git checkout # your branch
mkdir build
cd build
cmake ..
cmake --build .
```

This will generate the binary 'test' in 'build'.
